Requirements of the editor

For the modkit to not crash instantly you need to setup some things:

1. Launch the modkit with proton 9.0.4

go to steam and add UE4Editor.exe(under ModEngine/Engine/Binaries/Win64/ ) as a non-steam game

on steam, right click it and go to properties

under the compatibility tab check Force the use of a specific Steam Play compatibility tool

and select proton 9.0.4(only one i found to work, deviate at own risk)

2. Install some dependencies using protontricks

NOTE:

i am not sure if these are actually needed but i cant be bothered to test. having them shouldnt hurt either way as they are required for base contractors

install protontricks

open the program and select the Non-Steam shortcut to unreal engine that you made in a previous step

(you might get some errors/warnings here, you should be safe to ignore them)

select select the default wineprefix

select Install a Windows DLL or component

now select d3dcompiler_43,d3dx9,( vcrun2015 or vcrun2022 ) and dxvk and hit OK

you might get the same errors you faced earlier, you can again ignore them

if you get a warning about a checksum not matching you should still be fine as they happen often with frequently updated packages like vcrun

Open the editor

Open the Editor from steam

if unreal ask you to make a backup before opening, select the convert in-place option(it might be hidden behind a more options thingy)

you wont be able to launch the editor from the .uproject file, you must instead launch the editor manually or specify the path to the uproject in the launch arguments of the link in steam
