TFVR Map philosophy:

Map Interactions:

If an object looks like it could be interacted with, it should.

Lets take a glass bottle for example,
if we were to shoot the glass bottle, it should shatter or fly around.
Another example of map interactions is a bush,
where one would expect a short sound.

Simple interactions such as this can make your map feel "alive".

Map Immersion

If a given object typically does something, like a security camera moving around or tree branches moving in the wind, a mapper should consider adding these details.

Map Sound Design

Ambiance.
Leaves blowing in the wind, computers beeping, a river flowing.
these are all sounds that can ground you in a virtual world.

Map Particles

Particle effects in a map are a lot more important than many think,
they can add a lot more detail to something like a waterfall, making it seem more believable.
smoke or fog coming from somewhere, or some dust particles moving around also add to immersion in a map.
