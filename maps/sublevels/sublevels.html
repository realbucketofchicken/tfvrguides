Sublevels:

requires our modkit

We seperate Visuals and collisions from gamemode related things, like spawns and the payload
This is so we can reset the game state without having to reload everything(much slower)
Having sublevels is necessary for multi round maps

How do i configure sublevels?

its real simple:
Firstly you will enable the "Levels" window under Window>Levels

Now, create a new level, name it like the gamemode youre making
Drag this in the levels window

The level at the top is for things that dont change, The sublevel named after the gamemode youre making will be used to reload when a new round starts, so please make sure to not misspell it

Now you need to make sure that all gameplay related things are inside the gamemode sublevel
youre done!
